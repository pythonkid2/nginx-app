Welcome to Your NGINX Server!

Deployed via CodeDeploy

This project involves setting up a CI/CD pipeline using AWS, Jenkins, and CodeDeploy to deploy a simple NGINX web application. The pipeline ensures that whenever code is pushed to the repository, the web application is automatically deployed to a server, and the deployment is scalable with auto-scaling features to handle varying loads.
